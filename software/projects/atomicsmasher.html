Atomic Smasher

What does the program do?

The purpose of this program is to teach chemistry to children. The game is a particle simualtion game that involves the development of transportation systems using blocks to collide particles and form compounds. The program is developed in Java, and has been developed in a team with myself being the primary developer and graphics designer.

This is an example of a simple level, combining two particles, C and O to form Carbon Monoxide.

This is the main screen of the program.

Features:

The following features are present as of the latest release:

Integrated printable high-scores Efficient processing algorithm Various block pieces, including blocks that dynamically change position depending on compounds that pass through Block features included by default include split ends, combination blocks, diatomics and destination blocks Ability to pause and resume game Over 10 unique levels by default OOP (Object Oriented Design), with proper conventions used throughout Design allow easy level creation (simple instantiation of blocks and pieces) Allows multiple game profiles Unique splashscreen animations made in Motion X Includes background music Includes installer (DVD), *.jar file, and *.chm help file DVD cover with design is available Full JavaDoc documentation
