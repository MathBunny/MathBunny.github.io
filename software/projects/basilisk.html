What does Basilisk do?

Basilisk is a 3D game that allows the user to play a variation of the traditional snake game. Featuring over 10 game modes, Basilisk provides an entertaining twist on the classic game. The game features a moving snake that is composed of cubes, and you need to avoid hitting the edge or obstacles. Contrary to the original game, you can still hit your own body, however as you get larger you go faster and geting the apples becomes more difficult! This game is created using the Unity Engine, and scripted using primarily Javascript with a bit of C#. Feel free to look at some code on GitHub.

Features:

The following features are present as of version 2.0:

Multi-platform and optimized for mobile (iOS, Android) Features over 10 game-modes Allows local multiplayer with up to 6 players High-scores for local multiplayer Uses snake color distinguishing for multiplayer modes Lighting effects for snakes provides illuminating effect Random instantiation of cubes to make game more difficult Special apple powerups provide special abilities Includes game-mode variations (first person, birds eye view, etc.) Uses particle flare effects Creative interactive menus Customizable snake designs Hidden minigames Adjustable sound volume
