What does Decanting do?

Decanting is an Android puzzle game that allows you to play the classic mathematical game involving pouring water from one bucket to another. The challenge in the game is that you are given several buckets, and in each bucket you can only fill/empty it, or pour to another bucket. Given this, you have to fill any bucket to a specific capacity. View on Google Play Store here.

This is the game screen.

Features:

The following features are present as of the latest release:

Nice clean graphics Allows for interactivity with multi-touch gestures Includes high-scores with SQLite Comes with an infinite level generation, 10 levels pre-installed Has built-in solver using a caching BFS algorithm Applied OOP concepts throughout JavaDoc documentation Available on the Google Play Store
