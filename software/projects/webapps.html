Websites and Mobile Applications

The following are a series of websites and mobile apps that I have made in the past, both for personal purposes and for clubs at school.

WL Mackenzie CPT Website

While President of the William Lyon Mackenzie CI's Computer Programming Team, I created a website to manage team resources and post relevant links. Click here.

Features included:

Provides links to powerpoint resources Includes calendar with a list of events Nice graphics Features links to popular judges and contests
Android Fractals Generating App

This Android app, developed in Android studio, creates simple fractles. It currently renders using canvas by populating a bitmap and then showing them. To generate fractles, it uses Mandelbrot and Julias sets.

Features included:

Generates fractals by populating a bitmap and placing onto a canvas Utilizes mandelbrot sets for fractal generation Customizable colors, depth, quality

Feel free to view the website here.

WL Mackenzie Math Club Website

Simple website created for students to register on math contests. The website has been used over 500 times in the 2015/2016 to register for various contests.

Features included:

Embedded Google Forms registration page Registration includes automated email confirmation Includes math club points leaderboard
