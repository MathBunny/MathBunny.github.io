What does Convex Hull Visualizer do?

The Convex Hull Visualizer app allows you to construct convex hulls by plotting them on a plane. A convex hull (or convex envelope) is a set of points that is the smallest set which contains a set of points. View on Google Play Store here.

Simple convex hulls generated.

Features:

The following features are present as of the latest release:

Applied Graham Scan algorithm to compute convex hull Computes convex hull in O(nlogn) time complexity Allows use of multi-touch gestures to increase/decrease radius of nodes Permits customization of nodes & edges thickness and color attributes Applied OOP concepts throughout Available on the Google Play Store
