Arduino Robots

What do the robots do?

I have been part of several teams in which we constructed several different robots. I primarily focused on programming the robots - more specifically this includes maze solving robots and sumobots. The maze solving robot can traverse a maze using just an LDR sensor. Also, a sumobot was built to fight other robots and uses readings from LDR and Ultrasonic to navigate itself. Feel free to view the project on GitHub.

This is a birds eye view of the robot with its circuitry.

Features:

The following are some features in the robots:

Maze-Solving Robot (Arduino): Can traverse a maze using only one LDR sensor Maze-Solving Robot (Arduino): Features simple algorithm that works regardless of line intersections SumoBot (Arduino): Utilizes 4 ultrasonic sensors and 2 LDR sensors to strategically navigate and push opponent off from a ramp SumoBot (Arduino): Heuristics in algorithm allow for intelligent attacks and reactions to certain situations SumoBot (mBot): Simple SumoBot code for MIT's mBot programmed in Scratch & C to fight other robots using just one ultrasonic sensor and LDR
